面向对象、面向过程

面向对象：万物皆对象Object，通过对象去调用属性和方法
面向过程：把方法的参数按照一定的顺序执行完这些方法

对象的创建－字面量，原型，构造函数等各自特点

（你开发过什么什么插件）

字面量创建的方式使用constructor属性不会指向实例，而回指向Object

构造函数：
优点：在对象被创建时自动执行，函数名和类名相同，没有返回值和返回值类型，不能被显式调用

原型模式：
优点：共享
缺点：是省略了构造函数传参初始化的过程，使初始化的值都是一样的

继承 － 原型，原型链，es6继承 ，call，apply

原型：
所有引用类型都有一个__proto__(隐式原型)属性，属性值是一个普通的对象
所有函数都有一个prototype(原型)属性，属性值是一个普通的对象
所有引用类型的__proto__属性指向它构造函数的prototype

原型链：
当访问一个对象的某个属性时，会先在这个对象本身属性上查找，如果没有找到，则会去它的__proto__隐式原型上查找，即它的构造函数的prototype，如果还没有找到就会再在构造函数的prototype的__proto__中查找，这样一层一层向上查找就会形成一个链式结构，我们称为原型链

ES6继承：


call:调用一个对象的一个方法，用另一个对象替换当前对象,可以传递多个参数
apply:调用一个对象的一个方法，用另一个对象替换当前对象，最多只能由两个参数(this\argArray)

闭包－是什么，什么情况会出现，项目中用它处理什么

闭包：
即通过函数嵌套函数，内部函数引用局部变量实现变量不释放。广泛来讲，所有函数访问另一个函数内部变量的过程都可以称之为闭包

出现：
函数可以创建独立作用域
自由变量跨作用域取值
变量被函数引用时不会被释放

处理：
为了防止变量污染，但是用多了会内存泄漏

什么地方用：
闭包就是父函数给子函数传值，解决作用域问题。
在es6出现之后一般使用let

ajax交互的底层原理，如何封装ajax；ajax，axis，fetch区别对比

原理：
在用户和服务器之间添加了一个中间层(AJAX引擎)，使用户操作与服务器相应异步话，并不是所有的用户请求都提交给服务器

ajax:是一种用来创建快速动态网页的技术

Ajax 指的是 XMLHttpRequest

axios:
是基于promise用于浏览器和nodejs的HTTP客户端，本质上是对原生XHR的封装

fetch:
fetch是基于promise设计的。fetch的代码结构比起ajax简单，参数像jq ajax，但是fetch不是ajax的进一步封装，而是原生js，
没有使用XMLHttpRequest对象

封装ajax：
function ajax(options){
    var xhr = null;
    var params = formsParams(options.data);
    //创建对象
    if(window.XMLHttpRequest){
        xhr = new XMLHttpRequest()
    } else {
        xhr = new ActiveXObject("Microsoft.XMLHTTP");
    }
    // 连接
    if(options.type == "GET"){
        xhr.open(options.type,options.url + "?"+ params,options.async);
        xhr.send(null)
    } else if(options.type == "POST"){
        xhr.open(options.type,options.url,options.async);
        xhr.setRequestHeader("Content-Type","application/x-www-form-urlencoded");
        xhr.send(params);
    }
    xhr.onreadystatechange = function(){
        if(xhr.readyState == 4 && xhr.status == 200){
            options.success(xhr.responseText);
        }
    }
    function formsParams(data){
        var arr = [];
        for(var prop in data){
            arr.push(prop + "=" + data[prop]);
        }
        return arr.join("&");
    }

}

ajax({
    url : "a.php",  // url---->地址
    type : "POST",   // type ---> 请求方式
    async : true,   // async----> 同步：false，异步：true 
    data : {        //传入信息
        name : "张三",
        age : 18
    },
    success : function(data){   //返回接受信息
        console.log(data);
    }
})

深拷贝，浅拷贝

深拷贝:
就是复制了对象得地址，两个对象用得一个地址，所以当一个值改变得时候另一个值也改变（assign()）

浅拷贝:
只是把对象和值复制了过来，两个对象改变其中任意一个另一个不改变(jsonp()    json.stringify())
